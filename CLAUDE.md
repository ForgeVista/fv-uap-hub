# Claude Code Context (fv-uap-hub)

## Repo Purpose
Umbrella documentation for the UAP (User-Agent Pair) tier system. This repo helps users choose the right tier and provides migration guidance.

## Key Files
- `README.md`: Non-technical tier overview for consultants
- `AGENT-README.md`: Technical deep dive for AI agents
- `docs/tier-selection.md`: Decision matrix for choosing tiers
- `docs/migration-paths.md`: How to upgrade between tiers
- `docs/it-security.md`: IT approval talking points
- `docs/enterprise.md`: Tier 4 inquiries

## Guardrails
- Keep README.md simple and jargon-free
- Technical details belong in AGENT-README.md
- No secrets or API keys
- All links should be to public GitHub repos

- README.md: Written for non-developers (consultants, IT staff)

- docs/: Specific guides for common decisions
